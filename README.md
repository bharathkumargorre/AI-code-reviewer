# 🚀 AI Code Reviewer (Full Stack Project)

A full-stack web application built using **React (Vite)** for frontend and **Node.js + Express** for backend.  
It includes a login system and a protected dashboard with API integration.

## 📌 Features
- 🔐 Login system (mock authentication)
- 📊 Protected dashboard page
- ⚡ React + Vite frontend
- 🌐 Node.js + Express backend
- 🔄 Login → Dashboard flow
- 💾 Token stored in localStorage
- 🎨 Simple responsive UI (Tailwind CSS)

## 🛠️ Tech Stack
**Frontend:**
- React.js
- Vite
- React Router DOM
- Tailwind CSS

**Backend:**
- Node.js
- Express.js
- CORS
